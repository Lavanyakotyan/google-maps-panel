Google Maps – Place Details Panel

A React + Leaflet project that mimics Google Maps’ place details panel. Displays a map with a responsive side panel (desktop) or bottom sheet (mobile) showing place name, rating, address, timings, and a photo gallery. Timings are expandable, and the panel includes a loading state.

Features

Interactive Leaflet map with clickable marker.

Responsive panel: side panel on desktop, bottom sheet on mobile.

Static data for place details, ratings, photos, and timings.

Expandable/collapsible timings.

Loading skeleton for better UX.

Clean, reusable React components: PlacePanel, PhotoGallery, Timings, Loader.

Fully mobile-friendly and scrollable.

# Clone the repo
git clone https://github.com/Lavanyakotyan/google-maps-panel.git

# Navigate to project
cd google-maps-panel

# Install dependencies
npm install

# Run locally
npm run dev

Open in browser: http://localhost:5173
